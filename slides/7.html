Tech Issue #02

技術的実装課題：座標変換と表示方法

Coordinate Transformation & Rendering
1

起点検出
Origin Detection

上下左右の余白を除去 描画領域の原点座標を特定する。

2

比率計算
Scale Calculation

Illustratorの内部座標と、実際のWeb表示サイズとの拡大縮小比率を算出する。

3

座標正規化
Normalization

ページごとに異なる原点を統一座標系へ変換し、一貫した位置情報を生成する。

4

地図合成
Map Composition

文字情報を除外した「地図背景画像」と、透明な「SVGレイヤー」を重ね合わせる。

5

SVG描画
SVG Rendering

自由曲線(path)に沿ったテキスト描画や、領域内での折り返し(reflow)を表示する。

Input Data

x: 145.2, y: 302.5, rot: 15deg

Transform Matrix

matrix(0.8, 0.2, -0.2, 0.8, 10, 20)

Final Output

SVG TextPath

原文の塗りつぶしではなく、文字レイヤーそのものを置換することで視認性を確保
